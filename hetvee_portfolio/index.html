projects

Food Delivery App

Food Delivery App is a Django and DRF based Food Ordering and Tracking Web Application for User, Restaurant Owners and Delivery Agents. User can view the food dishes from various restaurants and can place the order. Restaurant owners can accept or reject the order and update the order status. Delivery agents can accept the order delivery and update the order status. The Payment is handled using Razorpay which is responsible for placing orders and managing payments and payoffs to restaurants and agents.

Django Grocery Store

Django Grocery Store is a Django based web application build for buying groceries online. It involves User and Vendor Authentication, Product registration and update, Cart options, Purchase Options, Stripe Payment Integration, Order, Invoice and Return Options.

LinkedIn Job Scrapper

LinkedIn Job Scrapper is a scrapper that searches jobs for Python Developer at Ahmedabad, Gujarat, India location on LinkedIn. The script implements Infinite Scrolling feature, so it is implemented using Selenium. It will extract the Job details like Position, Company Name, Location and Benefits.

QR based Authentication

QR based Authentication is Django based Web Application built for performing authentication using QR code. When a user registers, a new QR code will be generated. This QR can be scanned via any Authenticator apps like Google Authenticator, Microsoft Authenticator, etc. On scanning, a unique 6-digit code is generated every 30 seconds. When user logs in, an OTP page will be obtained, if user enters correct OTP, he will be able to login.

Razorpay with DRF

Razorpay with DRF is a Web Application which provides an example of creating Payment Link via Razorpay API, Verify Payment Link Signature, Fetch Payment Links, Webhooks, Configuration, Testing and, Validation.

Quotes Scrapper

Quotes Scrapper is a scrapper built using Scrapy which is very fast because of its async behaviour to scrape the quotes from a website by fetching quote, tag and author data from all pages to a json file. It also uses 3rd party libraries for rotating user agents and proxies.

BeautifulSoup Scrapper

BeautifulSoup Scrapper is a simple scrapper built using BeautifulSoup(bs4) to fetch the topic, link, quote and image of the quote from quotes website to a csv.
